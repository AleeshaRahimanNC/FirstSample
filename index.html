HTML Introduction

HTML is the standard markup language for creating Web pages.

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

HTML Tag List

To understand the HTML code better, below are some of the importand tags mentioned.

Tag Description
!DOCTYPE Defines the document type.
html The tag represents the root of an HTML document.
head Contains metadata/information for the document.
body Defines the document's body.
h1 Defines the most important heading.
h6 Defines the least important heading.
p The tag defines a paragraph.
em The tag is used to define emphasized text.
br The tag inserts a single line break.
strong Define text with strong importance.
ul Defines an unordered (bulleted) list.
ol Defines an ordered list. An ordered list can be numerical or alphabetical.
div Defines a division or a section in an HTML document.
b The tag specifies bold text without any extra importance.
i The tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.
li Defines a list item.
img The tag is used to embed an image in an HTML page.
